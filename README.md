# Shapes-Learning-App-for-Kids
Designed and developed an interactive educational app using Flutter, targeted at children aged 5–13, to help them learn and identify geometric shapes. The app categorizes shapes into 3-sided (triangles), 4-sided (quadrilaterals), and 5+ sided (polygons). Each shape is accompanied by an image and a kid-friendly description.

Shapes Learning App for Kids
Tools: Flutter, Dart, Android Studio, JSON, Image Assets, Animations
Platform: Android (Deployed via APK)

Description:
Designed and developed an interactive educational app using Flutter, targeted at children aged 5–13, to help them learn and identify geometric shapes. The app categorizes shapes into 3-sided (triangles), 4-sided (quadrilaterals), and 5+ sided (polygons). Each shape is accompanied by an image and a kid-friendly description.

Implemented a built-in quiz module that visually tests kids on shape recognition with multiple-choice questions and engaging feedback animations. The app features a colorful, intuitive UI and was successfully deployed as an APK using Android Studio for offline use.

Key Contributions:

Implemented dynamic shape listing and categorization

Developed animated quiz functionality for interactive learning

Handled asset management and responsive layout design

Ensured smooth performance and compatibility across Android devices
